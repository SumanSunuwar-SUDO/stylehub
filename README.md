# StyleHub

StyleHub is a full-stack fitness e-commerce application built with **Next.js** (frontend) and **Node.js/Express** (backend) with **MongoDB** as the database.

---

## Project Structure

- `client/` → Next.js frontend  
- `server/` → Node.js/Express backend  
- `.gitignore` → ignores `node_modules` and `.env` files  

---

## Prerequisites

- Node.js installed  
- MongoDB installed and running  
- npm or yarn package manager  

---

### 1. Clone the repository
